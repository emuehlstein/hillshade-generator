Generate Your Own

One command installs everything. Works on Mac and Linux.

# Install hillgen + all dependencies
curl -fsSL https://raw.githubusercontent.com/emuehlstein/hillshade-generator/main/install.sh | bash

Then generate a hillshade:

hillgen run --place "Crater Lake" --theme alpine-glacier --zoom 10-14
